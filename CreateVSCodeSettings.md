# VSCode 設定ファイル作成コマンド

プロジェクトルートに`.vscode`フォルダと空の`settings.json`を作成します。既に存在する場合は何もしません。

## 実行ルール

### ファイル操作

- `.vscode`フォルダが存在しない場合のみ作成する
- `.vscode/settings.json`が存在しない場合のみ空の JSON オブジェクト`{}`を作成する
- 既に存在する場合は何もしない（上書きしない）

## 実行手順

1. `.vscode`フォルダの確認と作成

   - `.vscode`フォルダが存在するか確認
   - 存在しない場合のみ作成する

2. `settings.json`の確認と作成
   - `.vscode/settings.json`が存在するか確認
   - 存在しない場合のみ空の JSON オブジェクト`{}`を作成する
