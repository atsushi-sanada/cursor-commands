# Unity Editorログ参照指示プロンプト

## プロンプト

UnityのEditorログファイルを読み取り、エラーや警告を分析してください。

## ログファイルのパス

以下のパスからUnity Editorログを読み取ってください：

```
C:\Users\%username%\AppData\Local\Unity\Editor\Editor.log
```

**注意**: `%username%`は環境変数なので、実際のパスは以下の形式になります：
- Windows環境: `C:\Users\<ユーザー名>\AppData\Local\Unity\Editor\Editor.log`

## 実行手順

1. **ログファイルの読み取り**
   - 上記パスからEditor.logファイルを読み取る
   - ファイルが存在しない場合は、ユーザーに確認する

2. **ログの分析**
   - エラー（Error）を優先的に確認
   - 警告（Warning）を確認
   - 重要な情報（Exception、Stack Trace等）を抽出

3. **問題の特定**
   - エラーの原因を特定
   - 必要に応じて、以下のCommandsを使用して分析・修正を進める：
     - `/AnalyzeBug` - バグの分析
     - `/FixBug` - バグの修正

## 出力フォーマット

以下の形式でログの分析結果を出力してください：

```
■ ログファイルのパス：
（実際に読み取ったパス）

■ 検出されたエラー：
（エラーメッセージと発生箇所）

■ 検出された警告：
（警告メッセージと発生箇所）

■ 分析結果：
（問題の原因や対処方法の提案）

■ 推奨アクション：
（次に実行すべき手順）
```

## その他

- ログファイルが大きい場合は、最新のエラーや警告を優先的に確認してください
- エラーが複数ある場合は、重要度の高いものから順に分析してください
- 必要に応じて、関連するソースコードファイルも確認してください
