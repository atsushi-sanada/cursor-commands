# cursor-commands

Cursor エディタで使用するコマンド集を管理するリポジトリです。

---

## 概要

Cursor エディタでの開発作業を効率化するための**コマンド**を集約しています。バグ修正、テスト実行、Git 操作、ドキュメント生成など、よく使う作業を標準化されたプロンプトとして提供します。

---

## セットアップ

コマンドファイル(.md)を以下のいずれかのディレクトリに配置します：

- **グローバルに使用する場合（推奨）**: `C:\Users\%userprofile%\.cursor\commands`
- **プロジェクト固有の場合**: `.cursor\commands` 以下に配置

※ディレクトリに直接 Clone するとメンテナンスが楽です

配置後、Cursor エディタを再起動すると `/コマンド名` で呼び出せます。

## 使い方

1. コマンドファイルを `.cursor\commands` ディレクトリに配置します（セットアップ参照）
2. Cursor エディタを再起動します
3. チャットで `/コマンド名` と入力して実行します

**例**: `/fix_bug` と入力してバグ修正を実行

---

## トラブルシューティング

- **コマンドが認識されない**: コマンドが正しく配置されているか確認し、Cursor エディタを再起動してください
- **実行結果が期待と異なる**: 各コマンドファイルの前提条件を確認してください
- **複数コマンドを組み合わせる**: `run_todo_tasks.md` を使用してタスクを分解してください
